Bem vindo ao aplicativo interativo  HidroSeries

Um aplicativo para acesso simplificado aos dados hidrológicos do Sistema Nacional de Informações em Recursos Hídricos (SNIRH) e geração de series historicas.

O aplicativo foi construído a partir da necessidade de acesso ágil por parte dos pesquisadores do Serviço Geológico do Brasil CPRM/SGB para aplicações em projetos voltados para as mais diversas finalidades: análises de séries históricas, previsões de cheias, análises de frequência de cheias, etc. A CPRM, além de ser o principal gerador de dados para a Rede Hidrometeorologica Nacional, operando cerca de 3/4 desta rede, também é um dos principais usuários desses dados em seus diversos projetos em Hidrologia Aplicada. Assim, um acesso ágil e simples aos dados do HidroWeb, mantido pela Agência Nacional de Águas pelo portal do HidroWeb, foi uma demanda gerada por esse uso intensivo de informação. Os mesmos dados que disponíveis por meio deste aplicativo também podem ser acessados pelo portal do HidroWeb http://www.snirh.gov.br/hidroweb/. O que diferencia esses conteútos é o formato dos arquivos e a forma de obtenção desses dados, porém, os dados são assencialmente os mesmos.

 COMO USAR O APLICATIVO:

  1. Navegando pelo mapa, você pode selecionar as estações para download clicando sobre elas. Clique uma vez e a estação ficará marcada com a cor vermelha. Clique uma segunda vez e ela será removida da selação e a cor mudará para laranja.
  2. Tendo selecionado as estações desejadas, clique em "Listar estações selecionadas" para ver quais estações selecionou.
  3. Clique em "Baixar dados das estações" - os dados serão salvos naquela seção e uma mensagem será apresentada, confirmando que os dados foram baixados com sucesso.
  4. Após esse procedimento, para que as funções das abas posteriores funcionem corretamente, recarregue a página apertando F5 ou o botão atualizar.
  5. Os dados também podem ser visualizados por meio de tabela dinâmica: aplique filtros, selecione a estação desejada e clique no ícone da coluna "Action". O mapa iterativa estará posicionado sobre a estação selecionada.
  6. No menu "Séries Históricas", clique em "Seleciona visualização" e selecione o código da estação que deseja visualizar, tipo de dado (cota ou vazão), início do ano hidrológico e tipo de série que deseja gerar.
  7. É possível fazer algumas seleções dos dados a serem gerados: estação, dados de cotas e vazões, início do ano hidrológico (especialmente importante para estimativas de médias, máximas e mínimas anuais), nível de consistência da série e o tipo da série histórica.
  8. Após clicar em "Gerar a Série", a série é apresentada em forma de tabela dinâmica e fica disponível para download. Além disso, a visualização gráfica também é disponível do sub item do menu "Visualiza gráfico".

IMPORTANTE: Os dados foram baixados só ficam disponíveis para aquela seção. Ou seja, ao finalizar a seção, todos os dados gerados são descartados e não ficam salvos na máquina. Para acessar novamente as séries históricas, elas deverão ser baixadas novamente.

IMPORTANTE: Os dados acessados aqui são os mesmos que podem ser baixados pelo portal do HidroWeb. Produzimos esta interface apenas para fins de acesso facilitado, de uma forma mais conveniente para as finalidades de pesquisa por parte da CPRM/SGB.
